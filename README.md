## Instructions

#### Server

1.Установить зависимости npm i  
2.Для корректной работы с CORS нужно создать параметры в .env - файле  
CLIENT_URL='http://localhost'  
CLIENT_PORT=3000  
3. npm run start  

#### Client

1.Установить зависимости npm i  
2.Для корректной работы с CORS нужно создать параметры в .env - файле  
REACT_APP_SERVER_URL = 'http://localhost'  
REACT_APP_SERVER_PORT = 8080  
3.npm start
